## 3. MANUSCRIPT STRUCTURE

### Title Page
- Title (concise, informative, <20 words recommended)
- Running head (<50 characters)
- Author names and affiliations
- Corresponding author (email, ORCID)
- Author contributions statement
- Data availability statement

### Abstract
- 250â€“350 words
- Structured: Aims, Location, Methods, Results, Conclusions
- 5â€“8 keywords (not repeating title words)

### Main Text
- **Introduction**: Context, knowledge gap, objectives/hypotheses
- **Methods**: Study area, sampling design, data collection, analysis
- **Results**: Present findings without interpretation
- **Discussion**: Interpret results, compare with literature, implications
- **Conclusions**: Brief summary of key findings (optional)

### Acknowledgements
- Placed after main text, before references
- Include funding information, permits, field assistance

## 4. IN-TEXT CITATION RULES (MANDATORY)

### Format: Authorâ€“Year system

**Single author:**
- Narrative: `Smith (2020) found that...`
- Parenthetical: `(Smith 2020)`

**Two authors:**
- Narrative: `Smith & Jones (2020) demonstrated...`
- Parenthetical: `(Smith & Jones 2020)`

**Three or more authors:**
- Narrative: `Smith et al. (2020) showed...`
- Parenthetical: `(Smith et al. 2020)`

**Multiple citations:**
- Chronological order: `(Brown 2015; Smith 2018; Jones 2021)`
- Same author, multiple years: `(Smith 2015, 2018, 2020)`
- Same author, same year: `(Smith 2020a, b, c)`

**Page numbers (when needed):**
- `(Smith 2020, p. 45)` or `(Smith 2020, pp. 45â€“48)`

### Special Cases
- Personal communications: `(J. Smith, pers. comm., 2020)` â€” NOT in reference list
- Unpublished data: `(J. Smith, unpubl. data)` â€” NOT in reference list
- In press: `(Smith in press)` â€” MUST be in reference list with journal name

## 5. REFERENCE LIST RULES (MANDATORY)

### General Requirements
- **Scope**: Include ONLY works cited in text
- **Order**: Alphabetical by first author surname, then chronological
- **Format**: Hanging indent (first line flush, subsequent lines indented)
- **Journal abbreviations**: Use standard abbreviations (ISO 4 or BIOSIS)
- **DOI**: Include when available as full URL: `https://doi.org/10.xxxx/xxxx`

### Ordering Rules
1. Single author: alphabetical, then chronological
   ```
   Smith A (2015)
   Smith A (2020)
   Smith B (2018)
   ```

2. Two authors: by first author, then second, then chronological
   ```
   Smith A & Brown C (2015)
   Smith A & Jones B (2018)
   Smith A & Jones B (2020)
   ```

3. Three+ authors: by first author, then chronological
   ```
   Smith A, Brown C & White D (2015)
   Smith A, Jones B & Taylor E (2018)
   Smith A, Green F, Blue G & Red H (2020)
   ```

4. Same author(s), same year: use letters a, b, c
   ```
   Smith A (2020a) First title alphabetically...
   Smith A (2020b) Second title alphabetically...
   ```

## 7. SPECIAL FORMATTING RULES

### Author Name Formatting
- Surnames before initials: `Smith AB, Jones CD`
- No periods between initials: `AB` not `A.B.`
- Use `&` before last author (not "and")
- Use comma before `&`: `Smith AB, Jones CD & Taylor EF`

### Title Formatting
- **Sentence case**: Only first word and proper nouns capitalized
- Correct: `Effects of grazing on grassland diversity in the Alps`
- Incorrect: `Effects of Grazing on Grassland Diversity in the Alps`

### Journal Name Formatting
- **Italicized**
- May use standard abbreviations or full name (be consistent)
- Examples:
  - Full: *Journal of Vegetation Science*
  - Abbreviated: *J. Veg. Sci.* (if abbreviations are used consistently)

### Volume, Issue, Pages
- **Volume in bold**: 31
- Issue in parentheses (optional): 31(2)
- Page range with en-dash: 245â€“258 (not 245-258 or 245â€”258)

### DOI Formatting
- Include when available
- Format as full URL: `https://doi.org/10.1111/jvs.12345`
- Place at end of reference (no period after DOI)

### Location Formatting
- City, STATE (if US), COUNTRY CODE
- Examples:
  - `New York, NY, US`
  - `Berkeley, CA, US`
  - `Cambridge, UK`
  - `Dordrecht, NL`
  - `Vienna, AT`

## 9. DATA AVAILABILITY & SUPPLEMENTARY MATERIAL

### Data Availability Statement (Required)
Must appear after Acknowledgements, before References.

**Examples:**
- `Data are available from the Dryad Digital Repository: https://doi.org/10.5061/dryad.xxxxx`
- `Data are available from the corresponding author upon reasonable request.`
- `Data are available in Supporting Information.`

### Supporting Information
- Appendices, additional tables/figures, datasets
- Referenced in text as `Appendix S1`, `Table S1`, `Figure S1`
- Each item needs a caption/description
- Must be uploaded as separate files during submission

## 10. TABLES & FIGURES

### Tables
- Numbered consecutively: Table 1, Table 2, etc.
- Title above table (brief but informative)
- Footnotes below table (use superscript lowercase letters)
- Cited in text in numerical order
- Previous publication requires permission and citation

### Figures
- Numbered consecutively: Figure 1, Figure 2, etc.
- Caption below figure
- Multi-panel figures use (a), (b), (c)
- High resolution (300+ dpi for photos, 600+ for line art)
- Color figures: indicate if essential for understanding
- Previous publication requires permission and citation
